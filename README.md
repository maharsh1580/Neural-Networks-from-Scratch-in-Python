# Neural Network From Scratch 🧠

A simple neural network implemented completely from scratch using Python and NumPy.

This project demonstrates the core concepts behind neural networks:
- Weighted sums
- Sigmoid activation function
- Forward propagation
- Error calculation
- Gradient-based weight adjustment
- Training loop

No deep learning frameworks were used. The goal was to understand the mathematics and mechanics behind a basic neural network.

## How It Works

The network takes input data, multiplies it with randomly initialized weights, applies a sigmoid activation function, calculates the error, and updates the weights through training.

## Tech Used

- Python
- NumPy

## Learning Goal

Built as a hands-on exploration of neural network fundamentals and the mathematics behind machine learning models.
